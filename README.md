# Загальці 🍺

Гумористичний 2D-платформер для Telegram Mini App. 5 хлопців із села Загальці
пробиваються до магазину по пиво.

Це **вертикальний зріз** — технічний скелет з плейсхолдер-графікою (кольорові
прямокутники), що доводить роботу всього ланцюжка: рух, стрибок, бій, ворог,
камера, досягнення цілі та збереження прогресу.

## Стек

- **Phaser 3** + **TypeScript** + **Vite**
- **Arcade Physics** (фіксований крок — детермінована симуляція)
- Збереження прогресу через **Telegram CloudStorage** (у браузері — localStorage)
- Структура «multiplayer-ready»: симуляція окремо від рендера, ввід оформлений як команди

## Запуск (локально)

```bash
npm install
npm run dev
```

Vite підкаже адресу (зазвичай http://localhost:5173). Відкрий у браузері.

**Керування (клавіатура):** ← → / A D — рух, ↑ / W / Пробіл — стрибок, J / K / F — удар.
**На телефоні:** екранні кнопки.

## Перевірка типів і збірка

```bash
npm run build      # tsc --noEmit + vite build -> dist/
```

## Як відкрити всередині Telegram

1. Створи бота через [@BotFather](https://t.me/BotFather) → `/newbot`.
2. Виклади гру на будь-який HTTPS-хостинг (для тесту зручний тунель:
   `npx cloudflared tunnel --url http://localhost:5173`).
3. У BotFather: `/newapp` (або налаштування бота → Mini App) → вкажи URL гри.
4. Відкрий бота в Telegram — гра запуститься у вбудованому вікні.

## Структура

```
src/
  main.ts            точка входу: Phaser + Telegram
  config.ts          усі ігрові константи (швидкість, стрибок, HP…)
  telegram.ts        обгортка Telegram SDK + збереження прогресу
  core/input.ts      ввід як команди (multiplayer-ready)
  scenes/
    BootScene.ts     генерація плейсхолдер-текстур
    GameScene.ts     рівень, камера, бій, ціль
  entities/
    Player.ts        герой
    Enemy.ts         ворог
```
